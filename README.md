# Chrome Profile Quick Launcher

这是一个利用 Windows 批处理脚本（.bat），在特定的 Chrome 配置文件（Profile）下快速唤起指定网页的工具集合。

## 设计初衷

在日常使用中，我们经常会在 Chrome 中登录多个不同的账号环境。当你正在使用配置 A（活动窗口）时，如果想立刻打开一个需要配置 B 账号环境的服务（例如特定账号下的 Gemini），手动去切换浏览器配置会打断工作流。

这些脚本的作用就是帮你绕过繁琐的切换步骤，直接带参启动目标配置文件的标签页，实现多账号环境的无缝穿梭。

## 配合快捷键食用更佳

这些 `.bat` 文件的核心价值在于**自动化联动**。虽然你可以双击运行它们，但更推荐配合第三方软件或硬件驱动器使用，实现物理级别的“一键启动”。

**应用场景示例：**

* **外设宏驱动**：例如通过雷蛇的“雷云”（Razer Synapse）软件，将键盘或鼠标的某个自定义侧键设定为运行指定的 `.bat` 文件。按一下物理按键，需要的特定账号网页就会瞬间弹出。
* **全局快捷键软件**：结合 AutoHotkey (AHK) 等快捷键工具，为不同的脚本绑定专属的键盘组合键。

具体的触发方式花样很多，你可以根据自己的桌面生态和操作习惯自行设计。

## 自定义与修改

如果需要添加新的网页或修改绑定的账号，只需用文本编辑器打开 `.bat` 文件进行简单的替换：

1. 修改结尾的目标网址。
2. 修改 `--profile-directory="Profile X"` 中的序号，以匹配你想要的 Chrome 配置文件（对应不同的 Google 账号）。
3. 确保脚本中的 `chrome.exe` 路径与你本机的实际安装路径一致。
